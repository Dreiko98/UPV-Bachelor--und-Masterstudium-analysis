## 🚀 Cómo Usar (3 Pasos)

### Paso 1: Instalar Dependencias (1 min)
```bash

npm install
```

### Paso 2: Ejecutar (30 seg)
```bash

```

### Paso 3: Usar (∞)

- Interactúa con las gráficas
- Expande los insights
